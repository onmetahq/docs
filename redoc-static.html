Introduction (1.0.0)

Onmeta provides the underlying infrastructure that allows DeFi companies, Web3 Gaming companies, NFT marketplaces & other Dapps to allow their end users to exchange fiat currency for crypto tokens and crypto tokens for fiat currency.

On/Offramp User Flow

Integration supported.

We support the following integration methods for our services i.e. both On and Off Ramp.

  • Widget Integration
  • API Integration In this document we will be discussing in detail the steps needed for widget integration in test module and then how to take the widget live. Before we dive into the integration subject, we will quickly highlight the key functionalities of our product in the following two sections.

Customizations Possible for Widget Integration.

In the widget there are a list of customizations that we allow that you can execute through your dashboard:

You can choose to have either On ramp or Off ramp or both

You can choose to use Onmeta's KYC module or integrate with your own KYC

You can choose to customize the fiat currencies and tokens that list in your widget

You can choose whether the user has to go through an email verification in our widget or not by choosing whether to pass the email ID of the user or not

You can choose the payment channels that you would like to use

You can choose the color theme of the widget from light and dark [WIP]

Webhooks

This section explains the steps to configure callback urls for receiving events about the transaction.

Configuring Webhook

Webhooks are configured in merchant dashboard.

Inside Developers section, webhook can be added and modified

Completed Order - Success

This callback will be triggered when the crypto coins are deposited to the given receiver address.

Headers
Key Value
Content-Type application/json; charset=utf-8
X-Onmeta-Signature {{signature-of-the-request}}

Signature is computed using Hash-based message authentication code (HMAC) using a secret key. The secret key is the API Secret present in your merchant dashboard.

Generating HMAC

The example implementation in Nodejs

let hmac = crypto.createHmac('sha256', apiSecret);
hmac.update(JSON.stringify(postBody));
let hash = hmac.digest('hex');
assert(hash == headers["X-Onmeta-Signature"])
